# Заметки по миграции

## Изменение модели Service

Поле `icon` было изменено с `CharField` на `ImageField` для возможности загрузки изображений.

### Что нужно сделать:

1. **Создать миграцию:**
   ```bash
   python manage.py makemigrations landing
   ```

2. **Применить миграцию:**
   ```bash
   python manage.py migrate
   ```

3. **Если были существующие данные:**
   - Старые значения в поле `icon` (CSS классы или пути) будут потеряны
   - Нужно будет загрузить новые иконки через админ-панель

### Структура папок:
- Загруженные иконки будут сохраняться в `media/services/icons/`
- Папка создастся автоматически при первой загрузке

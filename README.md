# Бюро Квартир - Лендинг

Профессиональный лендинг для агентства недвижимости "Бюро Квартир" в Петрозаводске.

## Описание проекта

Веб-сайт создан на Django с возможностью управления контентом через админ-панель. Позволяет редактировать услуги компании, примеры проданных объектов и статьи/новости.

## Технологии

- **Django 4.2.7** - веб-фреймворк
- **Django REST Framework** - для API (если потребуется)
- **Pillow** - работа с изображениями
- **Loguru** - логирование

## Установка

### 1. Клонирование репозитория

```bash
git clone <repository-url>
cd BuroKV
```

### 2. Создание виртуального окружения

```bash
python -m venv venv
```

### Windows:
```bash
venv\Scripts\activate
```

### Linux/Mac:
```bash
source venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Настройка окружения

Создайте файл `.env` на основе `.env.example`:

```bash
copy .env.example .env
```

Отредактируйте `.env` и укажите:
- `SECRET_KEY` - секретный ключ Django
- `DEBUG` - режим отладки (True/False)
- `ALLOWED_HOSTS` - разрешенные хосты

### 5. Применение миграций

```bash
python manage.py makemigrations
python manage.py migrate
```

### 6. Создание суперпользователя

```bash
python manage.py createsuperuser
```

### 7. Сбор статических файлов

```bash
python manage.py collectstatic --noinput
```

### 8. Запуск сервера разработки

```bash
python manage.py runserver
```

Сайт будет доступен по адресу: http://127.0.0.1:8000/

Админ-панель: http://127.0.0.1:8000/admin/

## Использование админ-панели

### Управление услугами

1. Перейдите в раздел "Услуги" в админ-панели
2. Добавьте новую услугу или отредактируйте существующую
3. Заполните:
   - Название услуги
   - Описание
   - Иконка (CSS класс или путь)
   - Порядок отображения
   - Активна/неактивна

### Управление объектами недвижимости

1. Перейдите в раздел "Объекты недвижимости"
2. Добавьте новый объект или отредактируйте существующий
3. Заполните:
   - Название объекта
   - Описание
   - Местоположение
   - Цена продажи
   - Тип недвижимости
   - Загрузите изображение
   - Отметьте "Продано" если объект продан
   - Порядок отображения

### Управление статьями

1. Перейдите в раздел "Статьи"
2. Добавьте новую статью или отредактируйте существующую
3. Заполните:
   - Заголовок
   - URL-адрес (slug) - генерируется автоматически из заголовка
   - Краткое описание
   - Содержание статьи
   - Загрузите изображение (опционально)
   - Дата публикации
   - Опубликовано/не опубликовано
   - Порядок отображения

## Производственное развертывание

1. Установите `DEBUG=False` в `.env`
2. Настройте `ALLOWED_HOSTS` с вашим доменом
3. Настройте базу данных (PostgreSQL рекомендуется)
4. Настройте веб-сервер (Nginx + Gunicorn)
5. Настройте SSL сертификат
6. Соберите статические файлы: `python manage.py collectstatic`
